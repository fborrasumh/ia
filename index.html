Ejecuta todas las apps de IA sin API key ni costes.
Tus datos nunca salen de tu ordenador.

1
Detectar tu hardware
Elegiremos el modelo más adecuado para tu equipo
2
Instalar Ollama y el modelo
Copia y pega en tu terminal
① Instalar Ollama
brew install ollama
② Descargar y arrancar el modelo
ollama run qwen2.5
3
Conectar con Ollama
Verificamos que Ollama está corriendo

⚠️ Estás abriendo la página desde HTTPS. El navegador bloqueará las peticiones a localhost por política de seguridad mixta.

Solución: ejecuta este comando y reinicia Ollama:

OLLAMA_ORIGINS=https://tudominio.com ollama serve
